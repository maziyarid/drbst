# DrBST Front-end

فرانت‌اند کامل وب‌سایت فارسی و راست‌به‌چپ دکتر شاهین باستانی‌نژاد، بر پایه کانسپت و سیستم طراحی موجود در مخزن `maziyarid/drbst`.

## وضعیت پروژه

- فرانت‌اند چندصفحه‌ای و ریسپانسیو کامل شده است.
- بک‌اند، ارسال واقعی فرم‌ها و اتصال به سامانه نوبت‌دهی عمداً پیاده‌سازی نشده‌اند.
- فرم تماس و فرم رزرو نوبت در حالت نمایشی کار می‌کنند و آماده اتصال به API هستند.
- حالت روشن و تیره با ذخیره تنظیم کاربر فعال است.

## صفحه‌ها

- `/` — صفحه اصلی
- `/about` — درباره پزشک و رویکرد درمان
- `/services` — خدمات جراحی و زیبایی
- `/blog` — مجله با جست‌وجو و فیلتر موضوعی
- `/contact` — اطلاعات مطب و فرم تماس
- `/appointment` — فرم سه‌مرحله‌ای درخواست نوبت

## قابلیت‌های فرانت‌اند

- رابط کاملاً RTL و فارسی
- ناوبری دسکتاپ و منوی کشویی موبایل
- حالت روشن و تیره
- طراحی ریسپانسیو برای موبایل، تبلت و دسکتاپ
- مقایسه تعاملی تصاویر
- آکاردئون پرسش‌های متداول
- فیلتر و جست‌وجوی مقاله‌ها
- فرم تماس با وضعیت موفقیت نمایشی
- رزرو نوبت سه‌مرحله‌ای با اعتبارسنجی، انتخاب زمان و صفحه بازبینی
- ساختار دسترس‌پذیر، فوکوس صفحه‌کلید و پشتیبانی از کاهش حرکت
- متادیتای مستقل برای صفحه‌های اصلی

## اجرای محلی

```bash
npm install
npm run dev
```

برای بررسی نسخه تولید:

```bash
npm run build
npm run validate:artifact
```

## اتصال بک‌اند در مرحله بعد

برای فعال‌سازی نسخه عملیاتی، این بخش‌ها باید به سرویس واقعی متصل شوند:

1. ارسال و مدیریت فرم تماس
2. استعلام ظرفیت و ثبت نوبت
3. تأیید پیامکی و اعلان به مطب
4. مدیریت مقاله‌ها از CMS
5. جایگزینی تصاویر نمایشی با تصاویر تأییدشده و دارای رضایت

هیچ‌کدام از این اتصال‌ها در وضعیت فعلی به‌صورت ساختگی شبیه‌سازی نشده‌اند.
